# Online Retail Exploratory Data Analysis with Python

![Python](https://img.shields.io/badge/Python-3.x-3776AB?logo=python&logoColor=white) ![EDA](https://img.shields.io/badge/Project-EDA-1F6FEB) ![Domain](https://img.shields.io/badge/Domain-Retail%20Analytics-B8860B)

## Overview
This project performs exploratory data analysis (EDA) on online retail transaction data to identify sales patterns, customer behavior, and product-level trends. The notebook is designed as a portfolio case study focused on practical business insights.

## Business questions
- Which products generate the highest demand and revenue?
- Which periods show peak sales activity?
- Which customer groups contribute most value?
- Where are there anomalies, returns, or outlier behaviors?

## Repository structure
- `online_retail_analysis.ipynb`: complete notebook analysis.
- `README.md`: project documentation.

## Tech stack
- Python
- Jupyter Notebook
- pandas, numpy
- matplotlib, seaborn

## How to run
1. Clone the repository.
2. Install dependencies:
   ```bash
   pip install pandas numpy matplotlib seaborn jupyter
   ```
3. Open the notebook:
   ```bash
   jupyter notebook online_retail_analysis.ipynb
   ```
4. Run all cells in order.

## Typical outputs
- Cleaned and transformed transaction data.
- Sales trends by time period.
- Product and customer contribution analysis.
- Visual summaries that support strategic recommendations.
